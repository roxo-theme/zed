## Installation

1. Clone this repository or download the `roxo-theme.json` file.
2. Copy `roxo-theme.json` to `$HOME/.config/zed/themes/`.
3. Apply theme in zed:
    - Open the command pallete
    - Search for theme selector: toogle
    - Select `Roxo Theme`
